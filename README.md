# sosaku

DSL for filtering JSON, for the individual components, see the following:

- [sosaku](./sosaku/README.md): The core Rust crate for evaluating expressions against a set of JSON values.
- [sosaku-cli](./sosaku-cli/README.md): A command-line interface for sosaku.
- [sosaku-py](./sosaku-py/README.md): Python bindings for sosaku.
